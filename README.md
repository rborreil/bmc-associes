# BMC Avocats - Template Site Vitrine

Ce projet est une reprise locale du design original `Lawify`, avec une appropriation initiale BMC Avocats.

## Structure

- `index.html` : page d'accueil
- `about.html`, `services.html`, `service-detail.html`
- `team.html`, `team-detail.html`
- `case-study.html`, `case-study-detail.html`
- `blog-grid.html`, `blog-grid-sidebar.html`, `blog-detail.html`, `blogs.html`
- `appointment.html`, `contact.html`
- `assets/css` : styles globaux + vendors
- `assets/js` : scripts globaux + vendors
- `assets/media` : images, icones, visuels

## Personnalisation prioritaire

- Coordonnees (telephone, email, adresse)
- Logo (`assets/media/logo.png`, `assets/media/footer-logo.png`)
- Mentions legales / politique de confidentialite
- Textes marketing (services, equipe, cas clients, blog)

## Notes

- Le rendu visuel est conserve au plus proche du modele.
- `blogs.html` est fourni pour compatibilite des liens du template.
